PathFinder AI: Gamified Career Quest for Youth
Youth Code x AI Hackathon 2026 Submission
Track: What Do I Even Do With My Life? (AI Can Help)
Live Demo: https://youth-code-x-ai-hackathon.vercel.app

The Problem
Career planning is broken for millions of young people worldwide, especially in emerging markets. Students face career indecision, weak resumes, poor interview preparation, and limited access to personalized guidance. Traditional tools are often boring, theoretical, and disconnected from real opportunities.

The Solution
PathFinder AI turns career development into an engaging RPG-style adventure powered by AI. By combining gamification with intelligent tools, we make career exploration exciting, practical, and motivating for youth globally with strong focus on the unique challenges faced in Nigeria and similar emerging markets.

Key Features
3D Career Explorer – Interactive map of real career paths and opportunities
AI-Powered Resume Analyzer – ATS compatibility scoring with visual feedback and smart improvement suggestions
Holographic Mock Interviews – Voice-powered practice sessions with real-time feedback
Salary Negotiation Simulator – Gamified boss-fight style negotiation training
Day-in-the-Life RPG Scenarios – Realistic workplace simulations
XP, Levels & Progression System – Level-locked features that reward consistent growth
Budget & Cost-of-Living Tools – Practical financial planning tailored to local realities

Tech Stack
React + Vite • Google Gemini 1.5 Flash • HTML5 Canvas + WebGL • Web Speech API • Web Audio API

Built For
This project was created to demonstrate how AI can solve real problems in education and youth employment making career readiness more accessible, engaging, and effective for young people around the world, starting with Nigeria.
