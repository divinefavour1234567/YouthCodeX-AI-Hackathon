PathFinder AI — Gamified Career Quest for Nigerian Youth

Youth Code x AI Hackathon 2026 Submission**  
Track 4: What Do I Even Do With My Life? (AI Can Help

Live Demo: [https://youth-code-x-ai-hackathon.vercel.app](https://youth-code-x-ai-hackathon.vercel.app)

The Problem
Career planning is broken for students in Nigeria. Indecision, skill gaps, bad resumes, and lack of guidance lead to frustration.

The Solution
PathFinder AI turns career exploration into an epic RPG adventure powered by AI.

Key Features
- Interactive Skill Tree with real Lagos/Nigeria opportunities
- Gemini AI Personalized Roadmaps
- ATS Resume Analyzer with visual improvements
- Voice-Powered Mock Interviews (Siri-style)
- Budget Simulator for financial planning
- Day-in-the-Life RPG Scenarios
- XP, Levels & Achievements system

Tech: React + Vite + Google Gemini + Canvas animations

Built to demonstrate how AI can empower the next generation of Nigerian talent.
