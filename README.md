PathFinder AI: Gamified Career Quest for Youth


The Problem:
Career planning is broken for millions of young people. Students face career indecision, weak resumes, poor interview skills, and limited access to personalized guidance. Traditional tools are boring, theoretical, and disconnected from real opportunities especially in emerging markets like Nigeria.


The Solution:
PathFinder AI turns career development into an engaging RPG-style adventure powered by AI. By combining gamification with intelligent tools, we make career exploration exciting, practical, and motivating for youth globally with strong focus on the unique challenges faced in Nigeria and similar markets.


Key Features:
3D Career Explorer with interactive maps
AI-Powered Resume Analyzer with ATS compatibility scoring
Holographic Mock Interviews with real-time voice feedback
Salary Negotiation Simulator (boss-fight style)
Day-in-the-Life RPG Scenarios
XP, Levels & Progression System
Budget & Cost-of-Living Tools tailored to local realities


Tech Stack:
React + Vite • Google Gemini 1.5 Flash • HTML5 Canvas + WebGL • Web Speech API • Web Audio API


Built For:
This project demonstrates how AI can solve real problems in education and youth employment by making career readiness more accessible, engaging, and effective for young people starting with Nigeria.
